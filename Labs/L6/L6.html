What exactly is a favicon?

A favicon is a small iconic logo that represents your website and they are located at the tab/browser address.

How are favicons created?

Favicons are created by using pixels to create the image. The pixels are created by using binary code, the binary code help create the colors of each pixel and when all the pixel come together they create an favicon that can be use for a website.

How did I create the favicon for this site?

I created this favicon for this site by using the pixelation tool on code.org. I created my favicon by having a 40 by 40 pixel and a 12 bit per pixels. Then I just typed the binary code for each of my pixel into the pixelation tool, until I finally finish it.

My icon is a rainbow pen. You can see my icon in the image above the heading that says "What exactly is a favicon?"
